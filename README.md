pyrest
============

Python base client for REST API

A simple generic REST APIClient that takes care of the encoding and packaging of parameters.

# Usage


This package is a library to be build on, if not use call to cal_api(...)
	
	
Dependencies
============

* six
* certifi
* mimetypes
